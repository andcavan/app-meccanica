# Fonti tecniche linguette e ghiere (2026-02-13)

## Linguette

- UNI 6604 (linguette parallele)
- ISO 773 (riferimento ISO per linguette parallele)
- DIN 6885 (equivalenza DIN)

## Ghiere KM

- ISO 2982-2 (locknuts with locking slots, tipo KM)
- DIN 981 (equivalenza DIN per KM)

## Ghiere GUK

- Tipologia commerciale `GUK` (assenza di riferimento ISO/UNI/DIN univoco nel dataset corrente).

## Regole applicate

- Gerarchia norme: `ISO > UNI > DIN`.
- Norma inserita in descrizione quando applicabile:
  - linguette: `UNI 6604`
  - ghiere KM: `ISO 2982-2`
  - ghiere GUK: nessuna norma univoca, mantenuto riferimento tipo commerciale.
