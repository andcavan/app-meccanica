# Fonti tecniche grani, spine e rosette (2026-02-13)

## Viti senza testa con cava esagonale (grani)

- ISO 4026 (piana): https://www.iso.org/standard/51620.html
- ISO 4027 (conica): https://www.iso.org/standard/51621.html
- ISO 4028 (cilindrica): https://www.iso.org/standard/51622.html
- ISO 4029 (coppa): https://www.iso.org/standard/51623.html

## Spine

- ISO 8734 (temprate/rettificate): https://www.iso.org/standard/55087.html
- ISO 8735 (temprate/rettificate con foro filettato): https://www.iso.org/standard/55089.html
- ISO 2338 (non temprate): https://www.iso.org/standard/55085.html
- ISO 8733 (non temprate con foro filettato): https://www.iso.org/standard/55084.html

## Rosette

- ISO 7089 (piane normali): https://www.iso.org/standard/61498.html
- ISO 7093-1 (piane larghe): https://www.iso.org/standard/65588.html
- Rosette elastiche: UNI 1751 / DIN 127-B (assenza riferimento ISO diretto equivalente nella struttura dati applicata).

## Equivalenze DIN usate nella gerarchia

- ISO 4026 -> DIN 913
- ISO 4027 -> DIN 914
- ISO 4028 -> DIN 915
- ISO 4029 -> DIN 916
- ISO 8734 -> DIN 6325
- ISO 8735 -> DIN 7979
- ISO 2338 -> DIN 7
- ISO 7089 -> DIN 125-A
- ISO 7093-1 -> DIN 9021
- UNI 1751 -> DIN 127-B (rosette elastiche)

## Regole applicate

- Gerarchia norme: `ISO > UNI > DIN`.
- Norma in descrizione prima della dimensione (`M` o `D`).
- Materiali inseriti secondo richiesta utente:
  - Grani: acciaio, inox.
  - Spine: varianti richieste m6/h8, con/senza foro filettato, acciaio/inox dove richiesto.
  - Rosette: piane normali/larghe ed elastiche, acciaio zincato/inox.
